## 安装与启动

- 执行安装命令
  
```bash
   $ yarn install --force or npm install
```
- 软件使用到原生的node模块，需要执行 [npm run rebuild](https://www.electronjs.org/docs/tutorial/using-native-node-modules)
  
- 启动

```bash
   $ npm run serve
```

- 打包
```bash
   $ npm run build
```
